O que são atributos nos elementos HTML?


O que é um atributo em HTML?


  • Todos os elementos de HTML podem conter atributos.
    Em alguns casos são mesmo obrigatórios.
  • São propriedades adicionais para configurar ou ajustar o comportamento dos elementos.
  • Os atributos são fundamentais na relação do HTML com o CSS e com o JavaScript.
  • Os tributos são definidos dentro da tag de abertura dos elementos HTML.

O elemento a (âncora) necessira do atributo href (Hiperlink reference).

Sem o atributo href, o browser apresenta o texto sem aspecto de link.

Atributos Globais - Podem ser usados em todos os elementos HTML

Existem atributos específicos para alguns elementos.

Atributos globais muito comuns: id e class.

  • id - identifica de forma única um elemento do documento HTML.
  • class - lista de classes de um elemento HTML.
